# 42Cursus-Cube3D

## Installation / Usage

This project is designed to work on Linux and macOS.

Clone repo:
```shell
git clone --recursive git@github.com:dspereira/42Cursus-Cube3D.git cub3d && cd cub3d
```

Build:
```shell
make
```

Usage example:
```shell
./cub3D maps/MapTest_001.cub
```
